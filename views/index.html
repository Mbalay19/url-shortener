URL Shortener

Transform your long URLs into clean, shareable links with our elegant microservice

Create Short Link

Enter any valid URL and we'll create a shortened version for you

API Documentation

POST /api/shorturl

Create a shortened URL from a long URL

Request Body:
{ "url": "https://example.com" }
Response:
{ "original_url": "https://example.com", "short_url": 1 }
GET /api/shorturl/:id

Redirect to the original URL using the short URL ID

Example:
GET /api/shorturl/1

Redirects automatically to the original URL

Error Handling

Invalid URLs return an error response

{ "error": "invalid url" }
